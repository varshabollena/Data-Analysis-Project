# Sales Data Analysis Report

## 1. Project Overview

This project performs a complete data analysis on a sales dataset using Python. The analysis includes loading the dataset, cleaning the data, calculating important sales metrics, and creating visualizations to understand patterns in the data.

The project demonstrates a complete data analysis workflow including data loading, cleaning, analysis, and visualization.

---

## 2. Dataset Description

The dataset used in this project is **sales_data.csv**.

It contains the following columns:

* **Date** – Date when the sale occurred
* **Product** – Product that was sold
* **Quantity** – Number of units sold
* **Price** – Price of one unit
* **Customer** – Customer
