# Data Analysis Report – Sales Dataset

## 1. Project Overview

This project performs a complete data analysis on a sales dataset using Python. The analysis includes loading the dataset, cleaning the data, calculating important sales metrics, and creating visualizations to understand patterns in the data.

The project demonstrates a full data analysis workflow using Python libraries.

---

## 2. Dataset Description

The dataset used in this project is **sales_data.csv**.

It contains the following columns:

* Product – Name of the product sold
* Quantity – Number of units sold
* Price – Price per unit
* Total_Sales – Total revenue generated
* Region – Location where the sale occurred

The dataset contains **100 rows and 5 columns** representing sales transactions.

---

## 3. Data Analysis Steps

The following steps were performed during the analysis:

### Step 1: Data Loading

The dataset was loaded into Python using the **pandas library.

### Step 2: Data Cleaning

The dataset was checked for missing values and cleaned where necessary.

### Step 3: Data Exploration

Basic information such as column names, data types, and dataset size was explored.

### Step 4: Data Analysis

Key metrics were calculated, including:

* Total revenue
* Best-selling product
* Sales distribution across regions

### Step 5: Data Visualization

Charts were created using **Matplotlib to better understand the sales data.

---

## 4. Visualizations Created

### 1. Bar Chart – Sales by Product

This chart compares total revenue generated by each product.

### 2. Line Chart – Sales by Region

This chart shows the distribution of sales across different regions.

These visualizations help identify which products perform best and where most sales occur.

---

## 5. Key Findings

From the analysis, the following insights were observed:

* The **Laptop** product generated the highest number of sales.
* Some regions contributed more revenue than others.
* Visualization helped identify product performance quickly.

---

## 6. Technical Implementation

The project uses the following Python libraries:

* **pandas** – for data loading and analysis
* **matplotlib** – for creating charts

The program performs:

* Data loading from CSV
* Data cleaning
* Metric calculation
* Chart generation
* Report output

---

## 7. Conclusion

This project demonstrates how Python can be used for basic data analysis and visualization. By combining data processing with graphical representation, useful insights can be derived from raw datasets.

The project helped improve understanding of data analysis workflows and visualization techniques.
